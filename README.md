# Tic Tac Toe Game
mini tic tac toe GUI game built with python using tkinter<br><br>

#### Clone the repo
```console
$ git clone https://github.com/shahan007/Tic_Tac_Toe
```

#### Set up the environment
```console
$ python -m venv venv
$ source venv/Scripts/activate
$ pip install -r requirements.txt
```

#### Start the game
```console
$ python run.py
```
